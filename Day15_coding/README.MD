### ðŸ’¡ Logic:

## âœ… Code (Using HashMap)
```java
import java.util.*;

public class SearchElement {
    public static void main(String[] args) {
        int arr[] = {1, 2, 3, 4, 5};
        int target = 4;

        HashMap<Integer, Integer> map = new HashMap<>();
        for(int i = 0; i < arr.length; i++) {
            map.put(arr[i], i);
        }

        if(map.containsKey(target)) {
            System.out.println("Element found at index: " + map.get(target));
        } else {
            System.out.println("Element not found");
        }
    }
}
```

## ðŸ’Ž Approach 3: Optimal Solution (Using Binary Search)

### ðŸ’¡ Logic:
- Sort the array (if unsorted).
- Use **Binary Search** to find the target element.
- Binary Search works by reducing the search space by half every iteration.

ðŸ‘‰ **Condition for Binary Search:** Array must be **sorted**.

## âœ… Code (Binary Search)
```java
import java.util.*;

public class SearchElement {
    public static void main(String[] args) {
        int arr[] = {1, 2, 3, 4, 5};
        int target = 4;

        int low = 0, high = arr.length - 1;

        while(low <= high) {
            int mid = low + (high - low) / 2;

            if(arr[mid] == target) {
                System.out.println("Element found at index: " + mid);
                return;
            }
            else if(arr[mid] < target) {
                low = mid + 1;
            }
            else {
                high = mid - 1;
            }
        }

        System.out.println("Element not found");
    }
}
```
